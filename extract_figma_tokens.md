# How to Extract Design Tokens from Figma

## Method 1: Figma Dev Mode (Recommended)
1. Open your Figma file
2. Click "Dev Mode" in the top right
3. Select any component/frame
4. Copy the CSS or design tokens from the right panel
5. Share the values with me

## Method 2: Figma Tokens Plugin
1. Install "Figma Tokens" plugin
2. Export tokens as JSON
3. Share the JSON file

## Method 3: Manual Extraction
Share these values from your Figma design:

### Colors
- Primary color (hex/rgb)
- Secondary color
- Success/Green color
- Warning/Yellow color
- Error/Red color
- Background colors
- Text colors

### Typography
- Font family (heading)
- Font family (body)
- Font sizes (h1, h2, body, etc.)
- Font weights
- Line heights

### Spacing
- Base spacing unit (usually 4px or 8px)
- Padding values
- Margin values
- Border radius values

### Shadows
- Box shadow values for cards
- Elevation levels

### Components
- Button styles (primary, secondary)
- Input field styles
- Card styles
- Badge styles

## Method 4: Screenshots
Take screenshots of:
1. Login screen
2. Dashboard
3. Onboarding form
4. Fraud detection results
5. Any other key screens

I can analyze the screenshots and extract the design patterns.
